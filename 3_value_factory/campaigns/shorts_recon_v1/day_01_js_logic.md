# Сценарий: Почему твой Node.js течет?

**Hook:** Твой сервер жрет 2 гигабайта памяти, хотя юзеров всего десять? Поздравляю, ты не понимаешь, как работают замыкания в JS.

**Body:** 
1. Ты вешаешь обработчик события внутри цикла или функции, которая живет вечно.
2. Внутри — ссылка на объект весом в пару мегабайт.
3. Сборщик мусора (GC) бессилен, потому что ты держишь ссылку мертвой хваткой.

**Insight:** Память в Node.js — это не бесконечный ресурс. Это твой главный ограничитель масштабирования.

**CTV:** Хочешь писать код, который не превращает сервер в кирпич? Подписывайся, разбираем инженерную базу.
