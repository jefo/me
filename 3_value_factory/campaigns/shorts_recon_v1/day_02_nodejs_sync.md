# Сценарий: JSON.parse убивает твой бизнес

**Hook:** Node.js — асинхронный? Только если ты ему не мешаешь. Один `JSON.parse` на 50 мегабайт — и твои пользователи ждут ответа вечность.

**Body:** 
1. Event Loop — это один поток. Если ты занял его тяжелой синхронной операцией, Node.js не может ответить другим юзерам.
2. Весь твой хваленый асинхронный ввод-вывод встает в очередь.
3. Ты платишь за облако, которое просто "висит" в ожидании CPU.

**Insight:** Производительность Node.js — это умение не мешать Event Loop-у делать его работу.

**CTV:** Не будь эгоистом, пиши асинхронно. Разбираем архитектуру высоконагруженных систем в следующих видео.
