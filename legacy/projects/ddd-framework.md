# Проект: Custom DDD Framework на Bun.js

Мой ответ на недостатки существующих решений в условиях эпохи ИИ и необходимости быстрого рефакторинга. Заменяет мне Nest.js.

## Ключевые особенности:
1. **Bun.js под капотом:** Изначально заточен под современный рантайм.
2. **DSL для LLM:** Фреймворк оперирует терминами классического DDD (Агрегаты, Инварианты, Use Cases). LLM видит эту структуру и интуитивно пишет правильный код.
3. **Функциональный подход:** Полный отказ от избыточных классов.
4. **Инновационный IoC:** Собственная система инверсии контроля и внедрения зависимостей без декораторов и конструкторов классов. Максимально лаконично.
5. **Чистая архитектура:** Домен из коробки изолирован от HTTP и баз данных.
